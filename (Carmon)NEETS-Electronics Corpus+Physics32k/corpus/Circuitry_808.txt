The  PIC16F84  is  an  excellent  choice  for  most 
simple microcontroller projects. This device is used 
in  many  current  PIC  designs  and  it  has  the 
following features: 
 
(cid:127) 35 single-word instructions (see Table 18.3) 
(cid:127) 1k × 14-bit EEPROM (Flash) program memory 
(cid:127) 68 × 8-bit general purpose SRAM registers 
(cid:127) 15 × 8-bit special function hardware registers 
(cid:127) 64 × 8-bit EEPROM data memory 
(cid:127) 1,000,000 data memory erase/write cycles
