Getting a printed circuit board made commer-
cially requires a design to be sent via the Internet to 
a service. It requires the following files in gerber 
format for a standard double-sided printed circuit 
board:
  1.  Top trace layout
  2.  Bottom trace layout
  3.  Silkscreen (if used)
  4.  Solder mask (if used)
  5.  Aperture file
  6.  Drill file
