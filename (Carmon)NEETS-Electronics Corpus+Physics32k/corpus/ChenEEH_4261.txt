The International Standard Organization’s (ISO) Open System
Interconnection (OSI) continues to be the de facto standard for
network reference model. All of existing networks and future
evolving networks continue to model their network based on
OSI’s reference model. This chapter introduces a general net-
work architecture using OSI and provides a more detailed
description of Local Network and Wireless Local Area Network.
WLAN has been extensively used in home/ofﬁce network and
there are increasing numbers of public hotspots worldwide. The
seamless interoperability between WLAN and mobile cellular
networks has been on the way. It is expected that utilizing IP
network to deliver various types of applications becomes a
norm. IP Multimedia Subsystem (IMS) is intended to deliver
ALL IP network architecture for all types of applications.
